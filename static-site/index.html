Cloud-based E-commerce REST API

A simple FastAPI service with products and orders. It uses SQLite for storage and HTTP Basic for login.

Source code: GitHub repository

Endpoints

MethodPathAuthDescription
POST/registernoneRegister a new user
GET/mebasicShow current user
GET/productsnoneList products
GET/products/{id}noneGet a product
POST/productsnoneCreate a product
PUT/products/{id}noneUpdate a product
DELETE/products/{id}noneDelete a product
GET/ordersbasicList orders
GET/orders/{id}basicGet an order
POST/ordersbasicPlace an order
PUT/orders/{id}basicUpdate an order
DELETE/orders/{id}basicDelete an order

How to run

git clone https://github.com/Alejandro2718/E-commerce-API-Project.git
cd E-commerce-API-Project
pip install -r requirements.txt
uvicorn main:app --reload

Then open http://127.0.0.1:8000/docs for the Swagger UI.
